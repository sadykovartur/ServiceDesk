# Lab 07 — Тесты + Docker Compose

## Цель
Сделать воспроизводимый запуск и минимум тестов, чтобы проект был "инженерным".

## Что сделать
1) docker-compose.yml:
   - postgres
   - server (optional)
2) Интеграционные тесты backend:
   - создание тикета
   - запрет доступа к чужому тикету (Student)
   - смена статуса (Operator)
3) Нормальный README: как поднять проект и прогнать тесты.

## Критерии приёмки
- docker compose up поднимает БД и (желательно) API
- dotnet test проходит
- тесты проверяют бизнес-правила (не только "200 OK")

## Команды проверки
- docker compose up
- dotnet test
