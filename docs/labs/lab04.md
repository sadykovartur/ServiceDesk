# Lab 04 — Интеграция: список и карточка тикета (React <-> API)

## Цель
Связать фронт с API: список тикетов + карточка тикета, обработка состояний.

## Что сделать
1) Настроить API клиент:
   - baseUrl через env
   - обработка ошибок
2) Подключить TanStack Query (рекомендуется) или альтернативу.
3) Реализовать Tickets List:
   - загрузка данных
   - фильтры: status, category, search (минимум 2 фильтра)
   - пагинация (минимум на уровне API или клиента)
4) Реализовать Ticket Details:
   - загрузка по id
   - отображение summary + comments block (пока можно без добавления)

## Критерии приёмки
- корректные loading/empty/error состояния
- данные берутся с backend
- фильтры работают
- навигация list -> details работает

## Команды проверки
- npm run build
- dotnet build
- docker compose up
